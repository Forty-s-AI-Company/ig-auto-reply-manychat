# Web Design Guidelines Skill

- Use this for pre-release UI review.
- Check consistency, readability, responsive behavior, and disabled UX.
- Every visible control must clearly be:
  - usable
  - disabled with reason
  - hidden
- Avoid visual states that imply missing engineering work.

## InboxPilot-specific release checks

- Simple release must not expose full-release dead ends without explanation.
- Mobile layouts must keep action buttons readable and non-overlapping.
- Workspace and channel scope indicators must remain understandable on narrow
  screens.
