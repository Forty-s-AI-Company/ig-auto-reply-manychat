# Design MD Skill

- Use this when a UI change affects shared language, hierarchy, naming, or
  visual consistency.
- Record durable design rules, not temporary bug notes.
- Keep the design language aligned across Dashboard, Inbox, Contacts, Channels,
  Automations, and Analytics.

## InboxPilot-specific rules

- Bright operational UI is the baseline unless an established section already
  uses another intentional style.
- Use consistent card borders, spacing, and status language across admin and
  end-user surfaces.
- Prefer clear Taiwanese Traditional Chinese copy where the product already uses
  Chinese UI.
