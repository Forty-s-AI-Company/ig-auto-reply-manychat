# AI_TEAM Skills

These files are project-local skill briefs for InboxPilot.

They are not marketplace plugins. They are reusable operating guides that
Codex, local models, and QA flows can read before working a topic.

## How To Use

1. Read the matching skill before starting a task in that area.
2. Treat the skill as a local operating contract, not as a generic style note.
3. Prefer project-specific constraints over broad framework defaults.
