# Automation template catalog

InboxPilot uses Traditional Chinese as the default UI language. This catalog is based on the Manychat Instagram Automation template modal observed in Chrome on 2026-05-30.

## Automation tabs

- 我的自動化: user-created automations, folders, search, filters, trash, list/grid controls.
- 基礎流程: fixed Instagram automation entry points such as Default Reply, new follower welcome, Conversation Starters, Story Mention Reply, and Main Menu.
- 序列流程: drip-style message sequences that send a series of automatic messages over time.

## Template categories

- All templates
- By goal
  - Grow your followers
  - Engage your audience
  - Drive traffic
- By trigger
  - Post or Reel comment
  - DM
  - Story reply
  - Live comment

## InboxPilot implementation notes

- Start From Scratch creates a blank comment-to-DM automation using the existing visual editor.
- Template cards create prefilled flows with trigger source, default keywords, suggested message copy, tags, fields, and short wait steps.
- Templates stay editable after installation; users land in the same visual editor.
